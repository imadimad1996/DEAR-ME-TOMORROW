# UI Spec

## Screens
- Main Menu (animated logo pulse + CTA stack)
- Merge Board HUD
- Orders slide panel
- Letters panel
- Rewarded ad flow modal
- Daily reward modal (stub)
- Starter pack modal (stub)
- Match mini-mode scene
- DevTest scene

## Merge HUD
- Top bar: energy, coins, gems, stars
- Generator buttons: Toolbench, Pantry
- Utility actions: Scrap Tool, Energy Ad, Room View, Mini Mode
- Booster strip: 8 slots (first 3 active)

## Motion Behavior
- Tooltip: quick appear with fade/scale intent
- Order completion: panel pulse + coin arc feedback
- Energy refill: fill tween + sweep + chime
