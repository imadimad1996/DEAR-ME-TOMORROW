# Art Style Guide

## Direction
- Semi-realistic painterly 2.5D.
- Warm coastal base palette with magical cyan/gold Echo accents.

## Brand Colors
- Gold `#F4C542`
- Deep Teal `#0F4C5C`
- Cream `#F6F1E9`
- Cyan Glow `#00E0FF`
- Coral `#FF6F61`

## Typography
- Elegant serif for titles.
- Clean sans for UI readability.
- Handwritten accent reserved for special narrative cards (future pass).

## Chain Progression Cues
- T1-T4: grounded materials.
- T5-T7: cyan/gold hints.
- T8: legendary ornate read + ambient shimmer.
