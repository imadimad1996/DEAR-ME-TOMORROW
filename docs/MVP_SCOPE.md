# MVP Scope

## Included
- Mobile-first HTML5 merge game with PixiJS.
- 1 fully playable room: Entrance Hall.
- 2 active chains: Wood + Culinary (8 tiers each).
- 4 chains in data (Wood/Culinary/Ocean/Tech), with Ocean+Tech disabled by config.
- Max 5 active orders.
- Echo visual overlay (single path, no branch choice).
- 5 letters.
- Energy spend + regen.
- Scrap blockers + clear actions.
- 3 boosters:
  - Extra Energy
  - Merge Joker
  - Time Skip
- Match mini-mode playable stub scene.
- Local analytics pipeline.
- Versioned save/load.

## Excluded
- Real IAP/real ad SDK integration.
- Multi-room full progression.
- Branching Echo narrative choices.
- Final production art/audio.
- Backend sync/liveops services.
